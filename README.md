**Perfume Landing Page**
**Project Overview**
This project is a responsive perfume landing page built with React and vanilla CSS. It features a modern UI with a responsive navbar, hero section, and product showcase designed to provide a clean and elegant user experience.

**Tech Stack**
React
Vanilla CSS
React Icons
Vite

Setup Instructions
npm install
npm run dev

Open:http://localhost:5173

**Features**
Responsive design for desktop, tablet, and mobile
Interactive mobile navigation menu
Dynamic product rendering using React
Clean and minimal UI design
Key Decisions
Used React components for better code organization and reusability
Used React state for the mobile menu toggle and scroll-based navbar behavior
Used CSS media queries for responsiveness across devices
Stored product data in a separate file and rendered it dynamically using .map()

**Future Improvements**
Add animations and transitions
Integrate shopping cart functionality
Connect to a backend/API
Add product filtering and search
